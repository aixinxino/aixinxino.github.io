辣酸土豆2023-04-142023-11-10

平常总会有些国外的网站想要进行访问，比如Youtube，包括我们目前最火的ChatGPT的官网，但是大多数情况下，我们去访问这些国外的网站总会被浏览器无情提示

办法千千万万，为非就是通过走代理的方式来进行访问，那么这就很麻烦了，

  • 我就想去Youtube上找一个视频学习，但是为什么我需要缴纳15￥的月费来透支我的时间呢，

那么这种时候我就在想，算了没必要把，或者说再找找免费的把。

要什么自行车呀？要啥自行车

节选–赵本山？

转折点来了！！

历尽千辛万苦，我还真就找到一个魔法机场，以极低的价格正在巧妙的改善这一类现象，

那么在这我一次性开通了3年（其实是没看到3个月和3年的区别……）

但是该说不说流量真的太多了…每个月我都会分配到5000个G的流量，那我每个月都用不完，那钱不久白出了吗，秉承着这一良好美德，我做出了一个违背…….

开个玩笑，那么在这里我将我的账号免费提供给大伙使用

操作起来也不麻烦分Android和PC两个端

这里介绍PC端，想知道Android端如何设置：**点击查看**

PC端的操作

第一步，下载好Clash for Windows.exe （关注公众号回复 “Clashwin”即可获取下载链接，放公众号是因为链接会挂）

那么下载好之后，我们打开页面，我使用的是中文版的，但是英文版的和这个大差不差，大伙自己对照着看操作就行了

这里Mac用户看过来（windows用户可以直接跳过这个部分）

Clash for Windows for Mac 是 Clash for Windows 作者的另一款作品。所以它的全名应该叫 Clash for Windows for Mac (很怪)

下载安装，地址放在公众号了，回复Clashios即可获得下载地址

下载好后打开.dmg格式的文件进行安装。

打开下载完成的 dmg 映像文件，将 Clash for Windows 添加到应用程序文件夹中。

如果macOS阻止运行该软件，请执行以下操作：

由于 macOS 默认情况下只允许运行可信任签名的应用，如果 macOS 阻止运行该软件，请打开 macOS 终端，在新建的终端 Shell 中输入：（没错就是你搜索你应用出现的终端两个字，打开复制粘贴之后就行了）

由于调用了sudo权限，你可能需要输入密码，会输出如下提示：

此时你需要输入密码，在 Shell 中输入的密码是不可见的，输入完毕后请按回车键。

如果您忘记了密码，或您的帐户没有密码，请在“用户与群组”偏好设置中添加或更改您的密码。然后，您便可以在“终端”中执行 sudo 命令。

在您键入密码时，“终端”不会显示该密码。如果您输入了错误密码或空白密码，此命令将不会执行，并且“终端”会让您重试。

那么接下来的操作双端都是一样的

那么我们先点击配置选项，然后找到上面的下载框，将URL 复制进去进行下载即可，最后成功后会得到一个yml文件，那么我们双击选择

URL：这个url可以是再别的网站自己购买，也可以去我公众号回复：“URL地址”（放公众号防止链接遭到攻击）

把你获得到的URL粘贴到配置框中间，然后点击下载更新，不出意外的话会多一个.yaml文件,出现了代表成功了！如果提示解析失败，请加群获取正确的URL或者自己仔细检查有没有遗漏或者多加了空格！

那么接下来我们来到代理服务，代理服务这里有两点一定要看清楚

  • 规则模式：简单来说就是你既可以进国内的站点，也可以浏览国外的站点，但是你的ip不会发生改变
  • 全局模式：简单来说就是你整个网络通过这个代理，那么你看国内的站点会有一点点卡顿，但是你的ip会发生改变

那么这就好理解了，如果你想去openai这种检测ip地址的网站，那你就选择全局模式，如果去的站点不会检查ip那么选择规则模式就好了

选择好地区之后我们来到主页打开系统代理即可，注意看到右上角的网速在动就证明已经搞好了

OK，我们选择全局模式来看看ip变化
